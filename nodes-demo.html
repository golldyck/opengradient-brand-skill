How It Works

Trustless compute
for AI inference.

Every model call on OpenGradient routes through verified node operators — you provide the hardware, we handle the protocol.

01 //
Serve Inference Requests
Your node loads AI models from the hub and processes incoming inference requests. Supports all OpenGradient-registered models including LLMs, vision, and time-series.
02 //
Proof-of-Compute Verification
Each inference is verified by a rotating committee of peer nodes using a lightweight ZK proof. Fraudulent results are slashed automatically by the protocol.
03 //
Earn OG Tokens
Rewards are distributed per verified computation epoch. The more reliable and fast your node, the higher your position in the request routing queue.
Requirements

What you need
to run a node.

No exotic hardware needed. A modern GPU and a stable connection are enough to start earning.

  • 💻
    GPU: 8GB VRAM minimum
    RTX 3080, A100, or equivalent. CUDA 11.8+ required. Multiple GPUs scale rewards linearly.
  • 🌐
    500 Mbps bandwidth
    Symmetric connection preferred. Model weights download once and are cached locally.
  • 💾
    500 GB NVMe storage
    For local model cache. Higher capacity means you can serve more model families simultaneously.
  • 🔐
    1000 OG stake
    Collateral ensures honest behaviour. Returned in full when you exit — slashed only for provable fraud.
