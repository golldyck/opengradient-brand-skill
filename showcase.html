Integration

Copy and paste

// 1. Add to head:
<script src="https://cdn.jsdelivr.net/gh/golldyck/opengradient-brand-skill@main/og-skill.js"></script>

// 2. Add to body tag:
<body data-og-build
  data-og-title="Your Title"
  data-og-tagline="Your description."
  data-og-cta="Get Started"
  data-og-cta-url="https://yoursite.com">
</body>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/golldyck/opengradient-brand-skill@main/og-brand.css">

<body class="og-scope">
  <h1 class="og-h1 og-text-gradient">Title</h1>
  <a class="og-btn og-btn-primary" href="#">CTA</a>
  <div class="og-card og-card-glow">...</div>
</body>
<script src="https://cdn.jsdelivr.net/gh/golldyck/opengradient-brand-skill@main/og-skill.js"></script>

<div data-og-logo="color"></div>
<div data-og-logo="cyan"></div>
What is included

Everything in one file

🎨
Brand Tokens
CSS variables for all colors, gradients, shadows and typography from the official OpenGradient brand guide.
⚡
Auto Page Builder
Add data-og-build to body and instantly get a full Hero, Nav, Sections and Footer.
🖼
Inline SVG Logo
No image files needed. The OpenGradient logomark is embedded as clean SVG.
📦
Zero Dependencies
No npm, no bundler. Pure HTML and CSS and JS served from jsDelivr CDN.
📱
Fully Responsive
All grids and layouts adapt to any screen size with fluid type and breakpoints.
🔧
Customizable
Use CSS variables to override any token and mix brand components into your design.
