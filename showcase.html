Integration

Copy and paste

<!-- 1. One script tag in <head> -->
<script src="https://cdn.jsdelivr.net/gh/golldyck/opengradient-brand-skill@main/og-skill.js"></script>

<!-- 2. Attributes on <body> -->
<body
  data-og-build
  data-og-title="Your Project Name"
  data-og-tagline="Your tagline here."
  data-og-cta="Get Started"
  data-og-cta-url="https://yoursite.com">
</body>
<link rel="stylesheet"
  href="https://cdn.jsdelivr.net/gh/golldyck/opengradient-brand-skill@main/og-brand.css" />

<body class="og-scope">
  <h1 class="og-h1 og-text-gradient">Hello World</h1>
  <p class="og-body">OpenGradient brand applied.</p>
  <a class="og-btn og-btn-primary" href="#">Launch</a>
  <div class="og-card og-card-glow">...</div>
</body>
<script src="https://cdn.jsdelivr.net/gh/golldyck/opengradient-brand-skill@main/og-skill.js"></script>

<!-- Full wordmark, white -->
<div data-og-logo="wordmark"></div>

<!-- Full wordmark, dark (for light backgrounds) -->
<div data-og-logo="wordmark-dark"></div>

<!-- Icon mark only -->
<div data-og-logo="mark"></div>
<!-- Brand overlay on any photo -->
<div class="og-img-overlay">
  <img src="your-photo.jpg" alt="..." />
</div>

<!-- Light overlay variant -->
<div class="og-img-overlay og-img-overlay--light">
  <img src="your-photo.jpg" alt="..." />
</div>
What's included

Everything in one file

🎨
Real Brand Tokens
CSS variables extracted directly from opengradient.ai — real colors, gradients, shadows, spacing.
🪪
Official SVG Logo
The exact OpenGradient wordmark and icon mark from the official site, embedded inline. No image files needed.
✍️
Geist Font
Geist + Geist Mono — OpenGradient's official font, auto-loaded from Google Fonts.
⚡
Auto Page Builder
Add data-og-build to <body> — full branded page generated instantly.
🖼️
Photo Brand Overlay
Apply OpenGradient brand colors as overlay on any image or photo with one CSS class.
📦
Zero Dependencies
No npm, no bundler, no config. Served from jsDelivr CDN. Works in any HTML file on any platform.
