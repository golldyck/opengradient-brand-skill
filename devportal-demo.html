01 · Quickstart

Your first inference in 60 seconds.

Install the SDK, set your API key, and run your first model on the decentralized network.

# Install: pip install opengradient import opengradient as og # Initialize with your API key client = og.Client( api_key="sk-og-your-key-here", network="mainnet" # or "testnet" ) # Run a text generation model result = client.infer( model="llama-3.1-8b-instruct", inputs={ "prompt": "Explain ZK proofs in one sentence:", "max_tokens": 128 }, verify=True # publish proof on-chain ) print(result.output) # → "ZK proofs allow..." print(result.proof_tx) # → "0x4a3f..." print(result.node_id) # → "node-sf-0842"
02 · Endpoints

POST /v2/infer

Submit a synchronous inference request. Returns the result plus an optional on-chain proof.

Parameter Type Required Description
model string required Model ID from the Hub (e.g. llama-3.1-8b-instruct)
inputs object required Key-value pairs matching the model's input schema. See model card for field names.
verify boolean optional When true, publishes a ZK proof to the OpenGradient contract. Adds ~200ms.
priority string optional One of low, normal, high. Default: normal
timeout_ms integer optional Max wait in milliseconds before returning 408. Default: 30000
200 OK 400 Invalid input 401 Unauthorized
